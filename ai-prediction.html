Architecture du Projet

Stack Technique

Plateforme analytique développée en Python. L'interface interactive est propulsée par Streamlit, offrant une expérience utilisateur fluide pour l'exploration de données énergétiques massives.

Python Streamlit Pandas & Numpy Plotly

Moteurs Analytiques Avancés

  • Analyse NILMDésagrégation de la consommation par appareil (modèle statistique).
  • Simulations What-IfImpact de l'intégration solaire, stockage batterie et efficacité HVAC.
  • Recommandations IAGénération automatique de stratégies d'optimisation énergétique.

Modélisation Prédictive

Algorithme XGBoost

Le cœur prédictif utilise un régresseur XGBoost, sélectionné pour ses performances supérieures face aux approches de Régression Linéaire, Random Forest et MLP. Le modèle capture efficacement les non-linéarités complexes de la consommation.

Features injectées :

Température Humidité Irradiation Solaire Vitesse du vent Heure & Date Jours Fériés

Explicabilité

Intégration de SHAP (SHapley Additive exPlanations) pour interpréter les décisions du modèle et comprendre l'impact exact de chaque variable météorologique sur la prédiction finale.

Plateforme Data & Prédiction

Import & Exploration des Données

La plateforme permet d’importer des données énergétiques depuis plusieurs sources : fichiers CSV, classeurs Excel ou bases de données SQL. L’utilisateur peut contrôler les données dans un tableau interactif avant de lancer l’analyse.

CSV Excel SQL Table Interactive

Visualisation Dynamique

Un moteur graphique permet de sélectionner plusieurs types de visualisations et d’explorer les tendances, les pointes de charge, les corrélations et les profils temporels de consommation.

Courbes Barres Aires Nuages de points Heatmaps Visualisations 3D

Prédiction à la Demande

L’utilisateur peut générer une prévision de consommation pour un instant précis en définissant la date, l’heure, la température, l’humidité et les autres variables disponibles. Le modèle transforme ces paramètres en une estimation directement exploitable.

Date précise Heure Température Humidité Variables météo

Simulation What-If Solaire

Un scénario comparatif permet de tester l’ajout d’une production photovoltaïque et d’évaluer son effet potentiel sur la demande nette prédite, les pointes de consommation et l’énergie soutirée du réseau.

Consommation de référence → Scénario solaire → Impact estimé
